# nav-ledger-loader

Navcoin Ledger App loader

# How to use

```
git clone https://github.com/aguycalled/nav-ledger-loader
cd nav-ledger-loader
python -mvenv venv
# On Mac/Linux:
source venv/bin/activate
# On Windows:
call venv\scripts\activate.bat
pip install fbs PyQt6
fbs freeze
fbs run
# Create binaries
fbs installer
```
